About Park Me

  • Park Me is a website focused on taking the pain out of parking.
  • You can quickly and easily find parking spots offered by other users.
  • You can also effortlessly post a parking spot you own for others to rent.
Finding A Parking Spot

  1. Selecting A Destination

    On the home page, you must select a destination (the geographical area you are interested in finding a parking spot). Most likely, you want to search in an area around your current location. To do this, simply click on the geolocate button, which looks like this:

    If you want to search a specific area other than your current location, use the destination text input box. As you type, a list of suggested locations will pop up. If the location you are interested in pops up, feel free to click on it so that you don't have to type out the whole thing yourself. Now that you have the desired location in the text box, you can instruct Park Me to search for parking spots. To do this, simply press the Enter key while you still have the input text box selected. Alternatively, you can press the search button, which looks like this:

  2. Filtering Only The Spots You Want

    If you want more control over the spots offered to you, select the Filter Options button, which uses the following icon:

    You can specify the maximum radius, in kilometers, from the provided search location to display parking spots. For example, if you are interested specifically in parking spots around a particular intersection then you probably want to limit the maximum distance to a very small value so that the map isn't cluttered with parking spots you don't care about.

    You can specify maximum prices, in dollars, of parking spots. This defaults to a value of 0 which indicates that you are interested in all prices for the given category. If you were to set the daily rate to a value of 15, then you would only see parking spots which cost $15 or less to rent for a day. This helps to remove expensive parking spots from the map which you aren't interested in, making it faster to find a spot which you are interested in.

  3. Picking A Spot

    On the map, parking spots are indicated by this icon:

    To view details about a spot you are potentially interested in, click on the parking spot icon. These details include prices about the rental and information about the user who is renting it out. If you like what you see, you can contact the landlord by sending an email to the address shown in the parking spot's information window.

Posting A Parking Spot

  1. Requirements

    If you wish to rent to other users, you must first create an account and login.

    • To signup, go here.
    • To login, go here.

  2. Posting A New Spot

    On the home page, click the Post New Parking Spot button, which uses this icon:

    Fill out all the fields displayed. The location/address refers to the location of the parking spot you are renting out (not necessarily your own address).

    The various rates refer to the price you are charging someone to use your parking spot. If, for example, you set the hourly rate to a value of 7, then you are indicating that someone can rent your parking spot for $7 per hour. By default, these prices are initially a value of 0 which indicates that you are not not interested in charging based on the given type of rate. For example, keeping the hourly rate value at 0 indicates to other users that you are not interested in charging by the hour.

    The parking type and information fields exist to provide other users extra information which they may be interested in. For example, some users may not want to park their vehicles outdoors. If you provide sufficient information about your parking spot in these fields now, you won't need to deal with other users frequently asking for additional information later.

  3. Controlling Posted Spots

    To edit parking spots you have posted in the past, head over to the settings page. To easily access the settings page in the future, click on the link in the navigation bar indicated by your own username next to this icon:

    In the settings page, you will see a list of parking spots you have posted. A parking spot can easily be marked as either Available or Rented by clicking on the Change Availability button for a particular parking spot. This button toggles between the two states. If a parking spot is marked as Available, then other users will continue to contact you about renting it. If a parking spot is marked as Rented, then the parking spot will not be visible to other users.
